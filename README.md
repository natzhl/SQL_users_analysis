# SQL_users_analysis
Данные получены из public dataset Google BigQuery thelook_ecommerce:  https://console.cloud.google.com/marketplace/product/bigquery-public-data/thelook-ecommerce?hl=en&project=causal-flame-283208

> TheLook — это вымышленный сайт по продаже одежды. Набор данных содержит информацию о клиентах, товарах, заказах, логистике, веб-событиях и цифровых маркетинговых кампаниях.

**Цель проекта** изучить набор данных с помощью sql-запросов, проанализировать пользовательскую активность, расчитать ключевые продуктовые метрики (MAU, DAU, Retention, LTV) и выявить точки роста для повышения эффективности бизнеса и увеличения выручки.
